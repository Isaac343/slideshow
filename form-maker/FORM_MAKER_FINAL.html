¿Que es Form Maker?

Se trata de una aplicación que permita al usuario generar formularios para sitios web. Programada en PHP y JavaScript.

Objetivos

General

Desarrollar una aplicación web que permita la creación de formularios para páginas estáticas de una forma fácil y rápida, reduciendo así en el tiempo invertido en las actividades que están involucradas.

Objetivos

Especificos

  • Establecer una estructura apta para el funcionamiento de la aplicación
  • Usar métodos, herramientas y paradigmas que permitan a la aplicación ser ligera y un funcionamiento óptimo.
  • Lograr una interfaz simple pero amigable al usuario.

    • Justificacón

    Existen herramientas que ayudan a mejorar la productividad de los desarrolladores al simplificar tareas o reducir el tiempo invertido en ellas. Esta aplicación fungiría en esta categoría al reducir el tiempo que es utilizado en la creación de formularios para las páginas web.
