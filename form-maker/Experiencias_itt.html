FORM MAKER

Generador de formularios

¿Que es?

Se trata de un software opensource a modo de aplicación la cual generará formularios de forma automática.

Objetivo

Facilitar el proceso de la creación de formularios para paginas web estaticas logrando así aumentar la productividad del desarrollador, pemritiendole dar mayor atencion a otros aspectos del desarrollo.

Caracteristicas

  • Gestión de resultados de formularios
  • Base de datos de texto plano
  • Interfaz amigable

    • Funcionamiento

  • Instalación
  • Configuración
  • Generación de formularios
  • Data Managment
